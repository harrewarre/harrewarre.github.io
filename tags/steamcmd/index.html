Valheim server on Linux

Here is a step-by-step guide for setting up a Valheim server on Linux. Having done it twice now I thought it would be a good idea to capture this somewhere so that next time I don’t have to look any of this up again. The only assumptions being made here is that your server is running Ubuntu of some semi-recent version and a basic understanding of Linux.
