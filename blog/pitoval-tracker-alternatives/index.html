Pivotal Tracker alternatives

Pivotal Tracker is an application designed to help teams manage their projects using agile methodologies. It allows users to create and prioritize stories, tasks, and epics, and collaborate in real-time with other developers.

Sadly, Pivotal Tracker is being shut down.

The website Goodbye Tracker - Welcome Trackers! is a successor to Pivotal Tracker, an award-winning agile project management tool. The site indicates that 10 teams are working hard to build the next iteration of this tool to allow teams to collaborate around a shared backlog of stories in real time.
